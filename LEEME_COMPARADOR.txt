🎯 QUÉ DESCRIBE ESTA DOCUMENTACIÓN
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  El "Comparador de Enmiendas" es una aplicación web que permite:

  ✅ Listar proyectos de ley de Puerto Rico
  ✅ Seleccionar dos versiones de un proyecto para comparar
  ✅ Visualizar diferencias textuales con colores:
     • �� Verde = Texto añadido
     • 🔴 Rojo = Texto eliminado
     • 🟡 Amarillo = Texto modificado
  ✅ Obtener análisis jurídico asistido por IA
  ✅ Subir nuevas versiones en formato PDF o Word

🛠️ STACK TECNOLÓGICO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Backend:        Django (Python)
  Base de datos:  PostgreSQL + pgvector
  Iconos:         Font Awesome 6.0

🔄 FLUJO DE USUARIO BÁSICO
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Usuario visita /comparador/
  2. Ve lista de proyectos de ley
  3. Click en proyecto → /comparador/<id>/
  4. Selecciona versión "Original" (dropdown izquierdo)
  5. Selecciona versión "Enmiendas" (dropdown derecho)
  6. Click en "Ver Diferencias" o "Explicar Cambios (IA)"
  7. Visualiza:
     • Tabla diff con cambios resaltados
     • (Opcional) Resumen jurídico IA
